Installation

Microsoft Windows

Laden Sie die setup-Datei herunter. Ein Installationsassistent begleitet Sie durch den Installationsprozess.

Linux (Debian basiert)

Um PhotoFilmStrip unter einem Debian-basierten Linux zu installieren laden Sie die deb-Datei herunter und starten Sie sie. Bei Ubuntu wird die Datei mit dem gdebi-Paketinstalltionsprogramm geöffnet. Alle Abhängigkeiten werden automatisch installiert. Anschliessend befindet sich eine Verknüpfung mit dem Namen PhotoFilmStrip im Grafik-Menü.

Andere Plattformen (vom Quelltext starten)

Der aktuellse Quellcode kann aus dem SVN repository ausgecheckt werden. Alternativ kann das source-tarball verwendet werden.

PhotoFilmStrip ist in der Programmiersprache Python (Version 2.5) geschrieben. Es ist nicht erforderlich den Quellcode zu kompilieren. Im "src"-Ordner befindet sich die Startdatei "photofilmstrip-gui.py" für die Benutzeroberfläche.

Voraussetzungen

Um PhotoFilmStrip zu starten sind folgende Komponenten erforderlich:

  • Python 2.5
  • GUI framework wxPython 2.8.0 oder später
  • Python Imaging Library (PIL)

Um einen PhotoFilmStrip zu rendern, sind folgende Programme erforderlich:

  • mjpegtools (VCD, SVCD und DVD Ausgabe)
  • mencoder (MPEG4 (XVid) Ausgabe)
